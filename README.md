# garinag/homebrew-tap

Homebrew tap для приложений [garinag](https://github.com/garinag).

## Установка

```bash
brew tap garinag/tap
brew install --cask review-reminder
```

Обновление:

```bash
brew upgrade --cask review-reminder
```

## Доступные пакеты

| Cask | Описание |
|------|----------|
| `review-reminder` | Трей-приложение macOS, напоминает про открытые GitLab merge request'ы. [Репозиторий](https://github.com/garinag/review-reminder) |

Cask-файл обновляется автоматически при каждом релизе `review-reminder` (см. `.github/workflows/release.yml` в основном репозитории).
